In this problem, you are given a line of text (use gets() to read a string line). Your task is to count the number of words in the input text.

Hint: fgets is more secure that gets

Input Format

A string line (use gets() function to read a whole line).

Hint: fgets is more secure that gets

Constraints

Max length is 256.

The input string will contain a-z, A-Z, 0-9, spaces, \t.

You should think about special cases and handle them. Your program must NOT crash under any input.

Output Format

i words

where i is the number of words in the input string

A word can start/include a-z, A-Z, 0-9

Sample Input 0

Hello World

Sample Output 0

2 words

Sample Input 1

Can you     count the   number  of words

Sample Output 1

7 words

Sample Input 2

   Be ware of special    cases    
   
Sample Output 2

5 words
