given an array of n integers, revers the array in place, then print it.
You can't use an extra array to solve the problem.

Input Format

first line contains integer n.
followed by n integers

Constraints

0 ≤ n ≤ 1000000
theres is no contrains on the values of the n integers.

Output Format

the n integers in reversed order

Sample Input 0

5

1 2 3 4 5

Sample Output 0

5 4 3 2 1
