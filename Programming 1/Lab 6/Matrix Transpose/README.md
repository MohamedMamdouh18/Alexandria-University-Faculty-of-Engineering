given matrix of dimension n X m find it's transpose

Input Format

first line contains two integers n and m which are the dimensions of the matrix
following this line n new lines. each line contains m integers

Constraints

1 ≤ n, m ≤ 1000
no constraints on integers of the matrix

Output Format

the transpose of the matrix.
m lines, each of them contains m integers

Sample Input

2 2
44 16 
-2 -42 
Sample Output

44 -2 
16 -42
