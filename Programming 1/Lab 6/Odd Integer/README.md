given n integers where each of them is repeated even number of times execpt one integer is repeated odd number of times. find this integer
hint : use xor.

Input Format

first line contains integer n
following line contains n integers.

Constraints

0 ≤ n ≤ 100000
no constraints on the values of the n integers

Output Format

a single integer x that is repeated odd number of times.

Sample Input 0

5
1 8 8 8 1 
Sample Output 0

8
Sample Input 1

3
4 4 1 
Sample Output 1

1
