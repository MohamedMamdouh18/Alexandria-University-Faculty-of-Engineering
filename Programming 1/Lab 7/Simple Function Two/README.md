Write a function digit(n , k) that returns the k th digit (from the right) in n (a positive integer). For example, digit(829, 1) returns 9, digit(829, 2) returns 2, 
and digit(829, 3) returns 8. If k is greater than the number of digits in n, have the function return 0.

Input Format

single line with two integers n and k

Constraints

1 ≤ n ≤ 1000000
1 ≤ k ≤ 6

Output Format

a single digit d

Sample Input

782383 2
Sample Output

8
Explanation

8 is the second digit form the right
