Write a recursive function that calculates the n th term of Fibonacci series. Compare it with the iterative version. Which one is faster? Why is that?

Note : Fibonacci of 0 = 0

Input Format

An integer n

Constraints

0 ≤ n ≤ 40

Output Format

Fibonacci of n = res

Where res is the return value of the function.

Sample Input 0

3

Sample Output 0

Fibonacci of 3 = 2

Sample Input 1

30
