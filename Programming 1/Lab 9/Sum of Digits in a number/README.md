Write a recursive function to calculate and return the sum of digits of a given number.

Input Format

An integer n


Output Format

Sum of digits of n = res

Where res is the sum of digits

Sample Input 0

9

Sample Output 0

Sum of digits of 9 = 9

Sample Input 1

20120250587024025

Sample Output 1

Sum of digits of 20120250587024025 = 45

Sample Input 2

-21001214

Sample Output 2

Sum of digits of -21001214 = -11
